## 🎧 Overview

A **fully functional Spotify-inspired Web Player UI** built using **HTML, CSS, and JavaScript**, designed to replicate the look and feel of the real Spotify application.

This project focuses on **UI/UX accuracy, responsiveness, and interactive playback controls**, providing a seamless music player experience.

---

## 🚀 Features

- 🎵 Play, Pause, Next, Previous controls  
- 📱 Responsive design (Desktop + Mobile view)  
- 🎧 Dynamic track switching  
- 📊 Interactive progress bar & volume control  
- ❤️ Like / Save / Download actions (UI simulation)  
- 🧭 Navigation (Home, Search, Library)  
- 🎨 Pixel-perfect Spotify-inspired UI
